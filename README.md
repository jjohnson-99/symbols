### Description

This plugin is in its very early stage. It will eventually be a popout
displaying definitions and references for functions and other symbols in the
target buffer, much like the symbols panel on GitHub. It utilizes treesitter on
the backend so requires neovim.

Its current state is just the bones, it has a functioning toggle and correctly
keeps track of the target window and correctly reads the buffer's language.

I expect to make slow improvements once a week or so.
